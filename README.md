# Scronth

Scronth is a social media platform where Everybody is kind!

## Features

- User authentication (sign up and log in)
- Post text and photos
- User profiles with followers, following, and friends
- Profile pictures
- Content filtering (AI-like detection for inappropriate content)
- Admin controls for moderation
- Classic 2004-2009 Web 2.0 styling

## Setup

1. Open `index.html` in your web browser
2. Sign up for an account or log in
3. Start posting!
